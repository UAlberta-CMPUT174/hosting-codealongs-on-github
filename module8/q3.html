Version 2:

For this version we will now write some useable functions. Write the following functions:

  • convert which takes in a string representing the currency to convert to and a float representing the amount of CAD to convert and returns a floating point which represents how much the given CAD would be worth in the currency
  • display_result which takes in a string representing the currency to convert to and a float representing the amount of CAD to convert and prints out a message stating what the given amount of CAD is worth in the given currency
  • main should now ask the user how much CAD they would like to convert, and print out the conversions for that amount in EUR, INR, and UZS

Scenario:

Mr. Ratburn has given his students a list of words. He has asked them to scan the list in order and move any word that starts with a vowel to the start of the list. At the end of this task, all words that start with a vowel would be listed before words that do not start with a vowel.Mr. Ratburn needs help with the answer key.

Task:

Create a program that:

  • scans a list of words
  • does the required task
  • prints the modified list of words

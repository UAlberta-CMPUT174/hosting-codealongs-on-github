Version 1

In this version, we will be creating the base of the game. You'll want the game to be able to:

  • create a game piece
  • initialize the starting position of the game piece to NOT ON BOARD
  • prompt the user to enter the size of the board
  • prompt the user to enter the number of sides of a die

We'll also want to create some UDFs:

  • create_board
    • This function should take in the board_size as an argument, and create and initialize the board
  • display_board
    • This function should display the initialized board. For example, if the board size is 5, a call to display_board should display: 
            ===========
      |0|0|0|0|0|
      ===========
  • roll_die
    • This function should take in the number of sides as an argument and return the value of a roll. The player should be prompted to press enter to roll, and the value of the roll should be randomly chosen.
  • And lastly, don't forget to include the main function!

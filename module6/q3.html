Version 2

In this version, we want to add upon version 1 by allowing the board to update the number of times a square has been visited, and display and update position the game piece on the board.

We'll need to add a new UDF for this:

  • update_board
    • This function will determine the new position of the game piece on the board by adding the roll to the previous position. It should also increment the number of times a game piece has visited each square on the board.
    • Make sure to modify display_board so that it properly displays the game_piece on the board!

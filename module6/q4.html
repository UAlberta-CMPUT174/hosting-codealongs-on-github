Version 3

In this final version of the game, we'll want to implement a way for the program to determine if the game should end. The game has two possible end conditions:

  • a space on the bard has been visited a MAX_VISITS times (for testing purposes, let's say the max is 3)
  • all spaces on the board have been visited at least once

Until the game ends, the program should keep updating and displaying the board, and prompting the user to roll the die.

We'll have a few final UDFs to define:

  • is_game_over
    • This function checks to see if one of the two possible end game conditions are true. If one of them is true, then the game should end; otherwise, the game should continue on.
  • display_result
    • This function displays a different message depending on how the game ended:
    • "Congratulations, you won." is displayed if all squares on the board has been visited at least once
    • "Unfortunately, you lost." is displayed if one of the squares on the board has been visited MAX_VISITS times

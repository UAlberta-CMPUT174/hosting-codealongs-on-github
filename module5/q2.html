Version 1

This version should contain a main function, which will ask the user for their username, and call a UDF to check if the given username is valid.

For this version we will only consider a subset of the username requirements. For a username to be valid it must have:

  • a single '@'' sign
  • a valid domain name

Additionally, the UDF should display a message to the user to let them know if their username is valid.

  • 'Valid username. Proceed to set up a password.' if it is valid
  • 'Invalid username' if not valid

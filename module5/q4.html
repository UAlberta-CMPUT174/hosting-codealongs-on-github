Version 3

This version will add upon the previous version by checking if the password that was entered complies with certain requirements. We'll also want to make adjustments to the main function so it will call these 4 new functions only if the username is valid and the passwords entered twice by the user match.

There will be 4 additional functions added in this version:

  • check_password_len
    • returns True if password has required length otherwise prints a message "The password must be at least 8 characters" and returns False
  • check_password_digit
    • returns True if password has at least one digit otherwise prints a message "The password must have at least one digit." and return False
  • check_password_capital
    • returns True if the password has at least one capital letter, otherwise it prints a message "The password must have at least one capital letter." and returns False
  • check_password_special
    • returns True if the password has at least one special character otherwise it prints a message "The password must have at least one special letter." and returns False

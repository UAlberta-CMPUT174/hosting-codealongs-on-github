Scenario:

Carol's trainer has given her a workout routine. The exercises in the routine are listed in a special format that is hard for Carol to understand. Here is an example:

Bench Press:5x4,45 secs means:
Perform 5 sets of Bench Press at 4 reps each Rest for 45 secs between each set.

Task:

Write a program that asks for an exercise code in the above format, translates it into a format that Carol is able to understand, and prints it.

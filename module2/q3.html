Scenario:

Carol wants to store the high protein smoothie in a rectangular container of the following dimensions:

  • Length = 21.3cm
  • Width = 26.7 cm
  • Height = 29.7 cm
Carol has prepared 8 liters of smoothie to last her for a week.

Task:

Write a program that calculates and prints how many containers Carol would need in order to store the smoothie.

Given:

  • Volume of a rectangular container = length x width x height
  • 1 cubic centimeter can hold 0.001 liters

Scenario:

Sookie has run into a problem recently. While working with the file, it has somehow become corrupted. The file now contains random amounts of white spaces in different areas.

Like so:

  Sarah     7 4   3

  Bob 9 2  7 5
  
  
  David     8  8 7   8
  Mary  6   5 7

Task:

Create a program that helps Sookie keep track of how many trout she has caught without exceeding any limits. Your program should print:

  • 'Trout cakes' if Sookie has enough trout.
  • 'Seafood medley' otherwise
